# 💅 Studio G Alberghini - Website

Este é o site institucional e catálogo de serviços do **Studio G Alberghini**, focado em sofisticação e experiência do usuário. O projeto apresenta uma interface moderna, responsiva e otimizada para agendamentos via WhatsApp.

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptado para celulares, tablets e computadores.
- **Catálogo de Serviços:** Seções organizadas para Manicure, Pedicure, Sobrancelhas e Depilação com preços.
- **Galeria Interativa:** Lightbox integrado para visualização de fotos dos trabalhos realizados.
- **Conversão Direta:** Botões de agendamento conectados diretamente ao WhatsApp.
- **UX Enhancements:** Botão "Voltar ao Topo" dinâmico para facilitar a navegação em páginas longas.
- **Localização:** Mapa interativo do Google Maps integrado ao rodapé.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Estilização com uso de variáveis, Flexbox e CSS Grid.
- **JavaScript (Vanilla):** Lógica do botão de topo, animações de scroll e sistema de lightbox.
- **Font Awesome:** Ícones sociais e de interface.
- **Google Fonts:** Tipografia moderna (Poppins).
